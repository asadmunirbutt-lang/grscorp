## STEP-BY-STEP FOR ARTICLES 4-15

### For each article:

1. **Find the HTML file** in your blog posts directory
2. **Open MEDIUM_INTRO_TEMPLATES.md**
   - Match the blog type (Guide, Research, Lifestyle, etc.)
   - Pick intro template A, B, or C
   - Customize with specifics from the article

3. **Extract content** from HTML file:
   - Skip CSS, scripts, headers, footers
   - Keep: `<h2>`, `<h3>`, `<p>` tags as regular text
   - Remove HTML formatting

4. **Format for Medium:**
   - Use Medium's native heading styles (H2, H3, H4)
   - Break long paragraphs into 2-3 sentences max
   - Add bullets for lists
   - **Bold** key points

5. **Add the outro:**
   - Use the standard template (see BATCH_1 guide)
   - Customize the [TOPIC] section
   - Link back to https://grscorp.us/blogs/

6. **Publish:**
   - Add 3-5 relevant tags
   - Click Publish
   - Update MEDIUM_PUBLISHING_TRACKER.csv with URL

## BONUS: MAXIMUM IMPACT TIPS

### Tip 1: Cross-Promote
- When you publish on Medium, share in your email list
- Share on social media (LinkedIn, Twitter, Facebook)
- Include backlink in all posts

### Tip 2: Optimize for Engagement
- Add 1-2 Medium-specific intros (not all blogs need them, but key ones benefit)
- Include a call-to-action mid-article on longer pieces
- Encourage readers to visit GRSCorp for the full resource

### Tip 3: Monitor Performance
- Which articles get most views? (clue for what your audience wants)
- Which drive most referral traffic? (best ROI)
- Publish more of what works

### Tip 4: Strategic Spacing
- Don't publish all at once (Medium's algorithm prefers consistency)
- 3-5/week is ideal - tells Medium you're active
- 5 days/week publishing = professional presence
